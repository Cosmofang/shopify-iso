## 页面结构

- **Foundations**：Polaris 颜色样式、文字样式、间距/圆角色板。
- **Components**：
  - Buttons（黑主按钮 / secondary / tertiary / plain / destructive 红 + loading/disabled）
  - Text field（默认 / 聚焦 / 错误+文案）
  - Select / Checkbox / Radio
  - Card / Section
  - Banner（info / success / warning / critical）
  - Badge（tones）
  - Toast、Modal（手动触发示意）
  - Page header、App nav

## 用法

1. 画新页面前，从 Components 页取组件，别自绘。
2. 颜色/文字用 Foundations 里的样式（对应 Polaris token）。
3. 交付前对照 [../00-built-for-shopify/pre-submission-checklist.md](../00-built-for-shopify/pre-submission-checklist.md)。
